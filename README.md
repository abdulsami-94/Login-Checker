Users enter a username and password, and the app instantly checks the input and displays a corresponding message.
Perfect beginner project for practising form handling, conditional logic, and DOM manipulation.

1. User enters a **username** and **password**.
2. The form listens to the `submit` event.
3. JavaScript prevents the page from reloading.
4. The script checks:

   * Are the fields empty?
   * Do they match the expected values (if you set them)?
5. The result is displayed inside the **output** label.
